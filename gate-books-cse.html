BEST OF GATE BOOKS CSE

Recommended Books of Computer Science (CS) for Gate Exam

We provide a very detailed list on books required to crack Gate Cse recommended by toppers so settle down grab some coffee and go through this list with your full attention.You may bookmark this website to get updated about further releases and notifications about GATE 2017 examination and latest books.\

ENGINEERING MATHEMATICS

B.S.Grewal for linear algebra and calculus is good and enough for Gate exam cse.


It is also good for generating functions and recurrence relations . ( Bessel's formula , Lagrangian Polynomial ).

Discrete Mathematics and Its Applications by Kenneth H. Rosen is probably the best Discrete Maths book

an excellent book for GATE preparation it has more problems given at the end of every chapter plus  

answers are also given ,the explaination is really good and easy to understand.

Relevant chapters are 1, 2, 6-11.

Trembley-Manoher includes good theorems in Group Theory.

Graph Theory : with applications to engineering and computer science by Narsingh Deo is good.

Try to go through all possible books and material for Graph Theory Graph Theory by Douglas West is also great book.

A first course in probability by Sheldon Ross - this is easy and efficient book for basic probability theory.

Lipschutz is a nice book which has clearly explained theorems, many solved problems, diagram are really.Discrete Mathematics,Probability Combinatorics, Set Theory & Algebra by Seymour Lipschutz (Schaum's Outline).

helpful. Further references are Pulak Kundu because of solved examples.

Erwin Kreyszig‘s book contains more number of solved problems it can be referred too.

Though numerical method is removed from gate syllabus but LU decomposition is still there.
Ref. Book : Introductory Methods in Numerical Analysis by S.S. Sastry but it is optional can be prepared online.

Book by K.D.Joshi on Discrete Maths has good derivations regarding enumerating the relations.

For Boolean Algebra and Group Theory again book by Trembley-Manoher is good.
Solved exercises are in book by Kolman buby Ross followed in some colleges as primary book.

You can see lot of solved examples in HK DAS book.

ALGORITHM

CLRS Introduction to Algorithm is best for Gate and other product based companies interviews ,

programming interviews even Google interviews question are asked from this book.

Many questions directly appears from the coreman in Gate Exam.

You need to be fully focussed while reading this books concentration is the key. Cormen is Good for Asymptotic Notation and Algorithm Design

Many says its the Programmers Bible so you should be ready to spend a day only reading it but don't go in 

deep read specific topic that are in gate syllabus.


  In case you are preparing for programming interviews this is the best book you must read.

Horowitz and Sahani is another good book for algorithms. If you need more advanced concepts refer Sedgewick               

Structure and Algorithmic Puzzles BY Narasimha Karumanchi is also good in concepts.

PROGRAMMING AND DATA STRUCTURE

C programming book by Dennis Ritchie is the best book and also most most recommended for c programming the author is the creator of C language.

You can refer to King book it contains essential concepts in elaborate ways.

  This introduction to the C programming language is based on the C ANSI dialect and is usable with all ANSI-compatible compilers and all operating systems.

   Data Structures and Algorithm Analysis in C++ is an advanced algorithms book that bridges the gap between traditional and Algorithms Analysis courses.

Programming Pearls – Bentley                 

C Programming:  – The complete reference by Herbert Schildt pointers are good especially in this book.

Yashwant Kanetkar Good for concepts – Programming easy language.

Balguruswami Good for Understanding -Programming if you need easy book.

DATA STRUCTURE

Classic Data Structures, Author’s Name: Debasis Samanta

Data structures and Algorithms – Aho, Hopcroft, Ullman    

Data structures Reference book I- Data structures using C & C++ by Langsam, Augenstein, tenenbaum   

Data Structures Schaum series from TMH Writer is Seymour Lipschutz   

DIGITAL LOGIC

Morris Mano book is all you would need for gate it contains all the things and bible for Digital Logic.


It is also the easiest and contains very simple questions of all.

Digital Circuits – Taub & Schilling 

COMPUTER ARCHITECTURE

MORRIS MANO book of C A O is good as it has some good theory and problems for addressing

modes , D m a , etc.

Pipe lining , Cache, and addressing modes are important topics and should be studied in detail.

Hamacher book good theory for Pipelining and detailed discussion in Cache memories.

Computer Architecture – A Quantitative Approach – Hennessy and Patterson is also good book used in IIT's.                           

Here another good book by William Stallings.                                                              

J.P.Hayes book also has good theory on Microprograming and Hard-wired control as well as pipelining

and Floating-Point arithmetic.

THEORY OF COMPUTATION

Michel Sipser book contains advanced concepts read it at last good for Un-decidability, Recursive and  Recursively Enumerable languages, NP-completeness.                                 

Ullman book contains good concepts on homomorphism and languages and most recommended for gate cse.             

Peter Linz book has all the basics and introduction to the theories of formal language and automata and

using mathematical expression for better understanding of the students focusing only on the concepts.                       

KLP MISHRA is also good. You can try Cohen and Martin after Ullman's book.

COMPUTER NETWORK

Kurose and rose book is good for application protocol .This book has all the necessary materials in relevant and fascinating way explaining how the internet works you should definitely try it.

This book has become the dominant book for this course because of the authors’ reputations, the precision of explanation and the quality of the art program.

Forouzan book is good for frame formats TCP and IP header formats and the exercises from forouzan are important

Tanenbaum book is best for data link layer,network layer,transport layer.Solve all the exercise take help of solution manual on esnips.com if required.

There have been questions in the past directly from Tanenbaum.                   

William Stallings book could be useful for certain topics For Numerical Problems                              

DATABASES

Navathe book is good for normalization part, ER diagrams. Navathe book is also good for physical database organization, transaction management and Normalization. good for SQL( & for general query languages) & normalization part. Navathe Relational Algebra, ER-Diagram, File Structure  

Korth book has all points required for GATE preparation good set of examples to understand database concepts like normalization, transactions, and SQL. Korth ER model & Transaction management Basics, Query languages, Transaction, examples        

Book by Raghuramkrishnan is good for physical database organisation RA, TRC , SQL. Use solution manual if required.           

OPERATING SYSTEM

Galvin book is the best book ever written on Operating systems. Few concepts like process (scheduling, synchronization), memory management concepts are kind of classic.                                                                                                                         

Modern Operating System by Andrew Tanenbaum is also good book for OS. It contains great number of problems. Also includes some of theory that is not included in Galvin’s book.                                                                                                              

Operating Systems by Stallings also contains tricky problem use solution manual if required. Good for Memory. Attempt problems in Stallings they are very important especially problems on memory management and virtual memory.                      

COMPILERS

Ullman’s book is only best book for better understanding of the subjects and sufficient for gate exam

so you can refer it along with solving previous year gate papers study topic from book then attempt the

questions this will give you confidence in GATE exam. It is also called as DRAGON BOOK .                                

APTITUDE

A Modern Approach to Verbal & Non-Verbal Reasoning by R.S. Aggarwal is of good for students that it

contains many practice questions.                                                                          

Quantitative Aptitude For Competitive Examinations, Author's by R.S. Aggarwal covers numerical ability            

SOFTWARE ENGINEERING 

Though it is removed form gate Syllabus.

Software Engineering: A Practitioner's Approach by Pressman

You can concentrate on few topics like Cyclomatic Complexity, Coupling and Cohesion, etc.          

An Integrated Approach to Software Engineering – Pankaj Jalote

WEB TECHNOLOGY

This is removed from Gate Syllabus but if you want to read w3schools is the best.

Arihant GATE – 15 Computer Science & Information Technology Previous Questions with Solutions.
